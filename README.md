## How to run

1. Clone the repository
```bash
git clone https://github.com/flameguy150/sudoku.git
cd sudoku
```

1. Activate venv
```bash
venv\Scripts\activate #Windows
source venv/bin/activate #macOS/Linux
```

1. Install dependencies
``` bash
pip install -r requirements.txt
```

1. Run Game
``` bash
python main.py
```
