## How to run

1. Clone the repository
```bash
git clone https://github.com/yourusername/sudoku.git
cd sudoku
```

2. Install pygame
```bash
pip install pygame
```

3. Run file
windows:
```bash
py sudoku.py
```
mac: 
```bash
python3 sudoku.py
```
